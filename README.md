# Number Guessing Game (C Language)

This is a simple console-based number guessing game written in C.

## How the Game Works
- The computer generates a random number between 1 and 100
- The user tries to guess the number
- Hints are given (greater or lower)
- The game ends when the correct number is guessed

## How to Compile and Run
1. Open terminal in the project folder
2. Compile the program:
   gcc guess_number.c -o game
3. Run the program:
   ./game
